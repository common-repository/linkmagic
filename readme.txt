== Installation ==

1. Upload `linkmagic` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to your Admin Panel, Settings and LinkMagic
4. Insert the API code provided at linkmagic.co/dashboard in the box and click Save Changes
5. If you use a caching plugin make sure to clear the cache
